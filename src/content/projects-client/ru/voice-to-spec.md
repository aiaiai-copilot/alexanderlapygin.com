## Задача

Клиент говорит на встречах быстрее, чем пишет. После часа звонка остаётся диктофонная запись и желание не проводить второй час за расшифровкой. Нужна цепочка: голос → структурированный текст → спецификация.

## Подход

Расшифровка — модель Anthropic, постобработка — структурирование по шаблону спецификации. Возможность редактировать промпты для конкретного клиента. Версионность — git под капотом.

## Стек

Astro для фронта, минимальный Node-бэк, хранение — git-репозиторий.

## Результат

Время на спецификацию сократилось с дня до часа. Клиент проверяет результат через diff-режим.
